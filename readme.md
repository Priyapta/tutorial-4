# Tutorial 4 Game Development

## Latihan Mandiri

Pada latihan mandiri level 2, saya menggunakan TileMap sebagai platform permainan dengan menambahkan collision agar karakter dapat berinteraksi dengan permukaan platform. Selain itu, saya membuat dua spawner yang bekerja secara acak (random) untuk memunculkan objek rintangan. Di akhir level terdapat spawner terakhir yang memunculkan final objek, sehingga pemain harus memutar jalur untuk menghindari rintangan tersebut dan mencapai kondisi kemenangan.
